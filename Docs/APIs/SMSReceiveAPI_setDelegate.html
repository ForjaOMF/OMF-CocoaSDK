SMSReceiver: setDelegate

Set the SMSRecevier delegate

(void) setDelegate: (id) theDelegate;

Parameters
theDelegate
  Specifies the delegate to use the receptionFinished function that will process the SMS messages

Return
None

Remarks
receptionFinished function need not be named must be declared as follows:

- (void) receptionFinished(NSMutableArray*) & messages_list);

where messages_list contains the list of received SMS messages.
